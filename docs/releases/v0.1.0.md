# L2MAS v0.1.0

First public MVP prototype release.

## Highlights

- English-first, open-source GitHub repository with Apache-2.0 licensing.
- Top-5 localized README entrypoints: English, Simplified Chinese, Korean, Spanish, and Japanese.
- Provider registry covering cloud, local, and mock providers.
- Standard capability contract for script, character, Live2D model, voice, lip-sync, motion, video, and quality review.
- Deterministic local mock pipeline that runs without cloud API keys.
- Local FFmpeg `video.compose` provider that can emit a real MP4 container when FFmpeg is installed.
- CI checks for JSON/YAML metadata, localization files, wording guardrails, provider coverage, ruff, Python compilation, unittest smoke tests, and Docker Compose config.

## Current Scope

This release is a verifiable prototype, not a production animation system. It proves the repository structure, provider routing contract, deterministic MVP path, and CI safety rails.

## Known Gaps

- Live2D runtime rendering is not wired to a real Cubism SDK adapter yet.
- Textoon, ComfyUI, Diffusers, TTS/STT, and voice conversion providers are registry entries or future adapters, not production integrations.
- A2A and MCP are represented by local compatibility layers for smoke tests while protocol packages and service implementations are stabilized.
- Kubernetes manifests remain v2.0 deployment direction, not a validated cluster install.
