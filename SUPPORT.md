# Support

L2MAS is an open-source prototype. Community support happens through GitHub.

- Use GitHub Issues for reproducible bugs, documentation gaps, and scoped feature requests.
- Use GitHub Discussions, if enabled, for design questions, provider experiments, and deployment notes.
- For suspected vulnerabilities, follow [SECURITY.md](SECURITY.md) instead of opening a public issue.

When asking for help, include the operating system, deployment mode, relevant provider registry entries, logs with secrets removed, and the exact command or workflow that failed.
